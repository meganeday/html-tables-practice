Common HTML Tags

Tag Description
<DOCTYPE!> Defines the document type to the browser; must include at the top of every HTML document
<a> Defines a hyperlink
<article> Defines an independent section of a document
<aside> Defines content aside from page content
<br> Defines a single line break
<button> Defines a clickable button
<body> Defines the document's body
<button> Defines a clickable button
<h1> to <h6> Defines HTML headings
<span> Defines a section of a document
